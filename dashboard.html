Downloads

Hypernex.Unity

Hypernex.Unity is the Unity client for experiencing the Social (Non/)VR Sandbox.

The Launcher will automatically manage and install the latest version of Hypernex.Unity for you.

If you prefer to manage the install yourself, you can download the Hypernex.Unity client instead.

If you are playing on an Oculus Quest/Pro, you can download the APK and sideload yourself.

Warning

The Android build is VERY UNSTABLE and crashes frequently.

Hypernex.CCK

The CCK (Content Creation Kit) for Hypernex. Use this to bring your assets into Hypernex.Unity!

Hypernex.Networking.Server

The Game Server software which hosts instances on your computer.

Port forwarding IS required and your IP Address WILL be broadcasted by hosting this.

This is recommended for advanced users who have CLI and Linux server management experience.

Info

Your hosting provider has stated that a GameServerToken is required. This requires ALL Game Servers to provide a Token (provided by your provider) for authentication.
